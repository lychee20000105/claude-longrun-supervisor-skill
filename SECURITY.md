# Security Policy

This Skill is designed for local automation. Review prompts and write scopes before running high-permission Claude CLI workers.

Default forbidden actions in the Skill include push, deploy, publish, permanent deletion, secret logging, production data mutation, `git reset --hard`, and `git clean -fd`.

If you find a security issue, open a private report through GitHub Security Advisories when available, or create a minimal issue without secrets or exploit details.
