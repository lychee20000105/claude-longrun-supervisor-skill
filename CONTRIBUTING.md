# Contributing

Contributions are welcome.

Rules for Skill changes:

- Update `SKILL.md` version notes for every change.
- Small change: increment by `0.01` where applicable.
- Feature-level change: increment by `0.1`.
- Architecture-level change: increment major version by `1`.
- Keep scripts dependency-light and runnable without project-level package installation.
- Do not commit secrets, tokens, local run artifacts, or private business data.
